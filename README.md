# Nuxt Minimal Starter

Look at the [Nuxt documentation](https://nuxt.com/docs/getting-started/introduction) to learn more.

## Setup
npm i - To install node-modules to the project.
npm run dev - for start project (.evn with API_KEY needed)

For project working you have to create .env file in the main derictory with the "NUXT_PUBLIC_API_KEY=...", where you have to insert your openweatherapi API_KEY. For more information about API and forecast visit "https://openweathermap.org/api"
